# Upstream Review Draft

## Summary

TradingAgents Control Room is an independent community-built observability UI around TradingAgents workflows.

## Why This Is Useful

- makes workflow execution easier to inspect visually
- provides a clearer interface for replay and operator review
- lowers the barrier for demos, debugging, and research sharing

## Scope

- UI and observability layer only
- sidecar integration for upstream TradingAgents execution
- documentation for local Docker and manual setup

## Non-Breaking Behavior

This project is designed to sit beside the upstream framework rather than replace it. The goal is low-risk review and optional adoption paths.

## Experimental Alpha Status

This repository should be treated as experimental alpha software. Known limitations are documented in `docs/known-issues.md`.
